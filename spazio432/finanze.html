⚠ Prima configurazione — crea la tabella su Supabase

La tabella finanze non esiste ancora. Incolla questo SQL nell'editor di Supabase (SQL Editor → New query) e poi ricarica la pagina:

create table if not exists finanze ( id uuid default gen_random_uuid() primary key, anno integer not null, mese integer not null, tipo text not null check (tipo in ('uscita','entrata','risparmio')), nome text not null, importo numeric(10,2) not null, nota text, target numeric(10,2), created_at timestamptz default now() ); alter table finanze enable row level security; create policy "accesso totale" on finanze for all using (true) with check (true);
